## Pygmalion Docs

Running the docs on your PC locally:

##### Requirements:
- `git`
- `nodejs`

##### Installation 

- Clone the repo:
```bash
git clone https://github.com/AlpinDale/pygmalion-docs && cd pygmalion-docs
```
- Run the docs using `retypeapp`
```bash
npx retypeapp watch
```
